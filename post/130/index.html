前面我说2019

2019-01-21 #帅张

stormzhang : 前面我说，2019 注定是个不寻常的年份，有人可能会觉得我随便说说，可能会说每一年都不寻常，那就跟大家简单说下，为啥 2019 不寻常吧：

  1. 首先第一个，2019 是新中国成立 70 周年，整数年份，每次都会特别重视，这点相信不用我多说了吧，因为这个涉及 zhengzhi 敏感，所以不能多说，只能说一句整数年份，很不寻常；

  2. 经历过 2018 年，下半年我就在星球说过，相信大家也都感受到了，大环境不好，经济不景气，裁员潮，很多人期待 2019 会不会有所改观，所以，2019 会是个经济风向标的关键年份。可能有些人不知道，现在高层已经预知到 2019 上半年会非常不好过，经济同样会很差，高层已经开会，在接下来的日子，所有媒体不允许发任何相关的，经济不好、裁员等之类的负面言论，这是你们不知道的，所以，上半年肯定会不好过，关键就看下半年经济能否有好转，2019，对经济的发展来说，会是蛮关键的一年。

  3. 之所以不寻常，就是因为 2019 会有不少变革，同时也会伴随着技术的变革，这其中没准就会孕育不少机会，很多人在这一年会不好过，但同时，会有不少人在这一年发掘一些机会，我自认为，互联网发展到现在，并没有大家想象的完全成熟饱和了，不少领域其实还是有不少机会的，这个后面我会单独抽时间来给大家说说。

总之，一起看就好了，你们了解我的，我会时刻关注新的趋势与变革的，以一种创业的心态面对未来的发展，以后也会多多在星球分享相关的心得，看能不能发掘一些机会。
