刘润 : 《刘润•5分钟育儿学堂》我和小米一起做饭，突然我的电话响了。一个陌生号码。我的手是湿的，所以用指关节按下接听和免提。电话那头传来一个训练有素的声音：&quot;您好，我是银行贷款中心的，请问……&quot;我用指节挂了电话。小米有点吃惊。我说这是个推销电话。小米问，为什么呢？万一是有事找你的呢？我觉得有必要给小米讲讲&quot;同理心&quot;，于是说：小米，你很喜欢看名侦探柯南。但是你知道，一个侦探最重要的能力是什么吗？小米问：是什么？是同理心。同理心，是妙手神探，和他的死对头，连环杀手，共通的底层能力。什么是同理心？同理心，英文叫empathy，是知道别人在想什么的能力。同理心，和同情心不同。同情心，是共享别人情绪的能力。同理心，是看透别人的理性（动机，逻辑，计划）；同情心，是共享别人的感性（快乐，痛苦，忧伤）。我问小米，比如刚才那个电话。如果确实是我的借记卡、或者信用卡所在的银行，打电话有事找我，你觉得他们会怎么说？小米想了想，说：你好，我是……是。我说：我是什么？小米说：我是招商银行信用卡中心，请问你是刘润先生吗？是的。我说。你会说：&quot;我是银行，请问……&quot;吗？小米想了想说，不会。这就像你打电话给你的朋友，你会说：我是小米，请问XXX在吗？你做自我介绍时，你会很自然说清楚自己是谁。招商银行打电话来，会说我是招商银行，不会说自己是银行；中国移动打电话来，会说自己是中国移动，不会说自己是手机公司；海尔公司打电话来，会说自己是海尔，不会说自己是家电工厂。因为，招商银行，中国移动，海尔公司，才是可以用来表明身份的自我介绍。这是正常的逻辑。一个电话打来说，&quot;我是银行贷款中心&quot;的。这个介绍，没有表明身份，却想从你那里获得信任感。说明她很可能并不属于任何一家银行。她属于任何一家银行，都会说&quot;我是XX银行&quot;，而不是&quot;我是银行&quot;。那她是什么？她可能是推销，甚至是骗子。所以我挂了电话。站在对方的动机、逻辑、计划的角度思考，这就是同理心。小米说：哇，同理心这么厉害。我对小米说，挂掉推销电话，可能只是同理心的最微不足道的价值。妙手侦探要靠同理心破案。一个人推门进来，他的动机、逻辑、计划是什么？然后呢？再然后呢？连环杀手要靠同理心作案。一个人推门进来，他的动机、逻辑、计划是什么？我这样，我那样…同理心，是一种非常强大的能力，它是很多能力的元能力。演讲能力，沟通能力，销售能力，管理能力，辩论能力，谈判能力，等等，都是从同理心上长出来的二级能力。有同理心的人，可以通察人心。慢慢练习吧，小子。2019-08-11点赞数:110评论区：Emily.Z🍭 : 小米从小训练有素，长大不得了… 菊姐 : 这样的家教，没有学校教的了这样的孩子 郭惠 : 从小每天接受中国最贵的商业顾问之一的辅导，是怎样一种体验[偷笑] 王清兮 : 记得儿子三岁的时候，我没有管他懂不懂就是单纯跟我学：“像不等于是！”“别人做错不是我做不对的理由”。小家伙一开始只是大舌头的发音学，然后我就讲给他听。有一天他突然说，妈妈你看这里很像我的幼儿园，但不是。今年孩子9岁了，他在对事物逻辑分析上不是同龄孩子的思维了。有自己的认知和看法。我给他讲我的工作从来都是像和大人讲话那种，6岁那年我和他讲我的促销模式，小家伙一本正经的说，“妈妈，你让别人挣到钱你才能挣到钱。”[呲牙]教孩子模仿，不如教孩子思考！刘润老师在教孩子“小事儿背后大逻辑”这值得我们做父母的学习！今天我也学到了同理心和同情心的巨大差别，受益！ 王清兮 : 学校只是基础知识教育，不能期待很多。真正差异化的教育来自家庭。 深蓝_成&amp;渝 : [强] 小张 : 把自己放进别人的鞋子里，用对方观点说服自己 先归纳要点，再用演绎法演绎对方的观点。 张小胖 : 这也算是一种筛选顾客，能接着往下聊的才可能是目标对象。

孙瑞希-生涯咨询师 : 作者 | 孙瑞希：生涯咨询师，个人成长教练，今日头条签约作者，青云计划外部评审团成员。【01】任达华复工，拒绝赔偿引人深思在遭遇行刺18天后，香港演员任达华对外宣布了自己的决定。他表示，对于赔偿，“不用了”，因为这不是钱可以衡量的事情。昨天（8月9日），有圈中人在社交网络晒出任达华的最新工作照，他已经复工了。7月20日，任达华在广东参加商业活动时突然遭到歹徒的袭击，伤及手指与腹部。他先后进行了两次手术，还需要几个月的调养，才能恢复健康。根据警方通报，行刺者存在精神障碍。任达华知晓情况后表示不会追究责任，也不要求赔偿。这件事情引发网友的热烈讨论，有人认为任达华作为公众人物有气魄、有格局；有人质疑，既然是商业活动，完全可以找主办方索赔，为什么不主张自己的权益呢？对于公众人物来说，时间和精力是极为宝贵的资源。在这起事故里，行刺者有精神障碍，向他索赔，需要经历复杂的精神鉴定过程；向主办方索赔，会对自己今后的商业合作造成影响。既然如此，不如放弃，这也许是最好的选择。任达华拒绝赔偿，体现的不仅是宽容和格局，短短三个字“不用了”，足以体现他决策的智慧：不把时间和精力浪费在消耗自己的事情上。诺贝尔经济学奖得主斯蒂格利茨教授在《经济学》一书中说：“如果一项开支已经付出并且不管作出何种选择都不能收回，一个理性的人就会忽略它。这类支出称为沉没成本。”生活中，人们难免会遇到糟糕的事情，造成严重的损失。如果你继续纠缠下去，你不但要为之前的投入买单（沉没成本），还会浪费更多的资源扩大投入，这个扩大的投入叫做追加成本。从经济学角度来说，如果你已确定继续在这件事中纠缠毫无价值，最明智的选择就是马上离开。【02】不舍弃，就可能拖垮你2009年冬天，我经历了一场车祸，这让我第一次深度思考“沉没成本”这个概念。那天，我和先生停车等待红灯放行。远处一辆黑色奥迪不知是溜号还是把油门当成了刹车，它飞速开来，狠狠地撞向了我们。强大的冲击力下，我们的车又撞向了前面的箱货。前后夹击，车损严重，我和先生也都受了伤。毫无悬念，奥迪车负全责。去交警队办理手续那天，我刚拿到事故鉴定书，肇事方就被bianyi警察当场抓捕。原来，肇事方伙同他人涉嫌重大案件。我当时就傻眼了，那我的损失谁来赔偿？经保险公司鉴定，车辆定损4万余元。肇事方的案情严重，上了我们当地发行量最大的报纸。我也从律师那里获悉，如果我想通过民事诉讼拿到车辆赔偿款的话，过程极为漫长。简单来说就是：我只能等到肇事方的刑事案件审理完后，才能追究其民事责任。和律师见完面后，我开始研究“沉没成本”，即已经付出的、不可回收的成本。我合计了一下民事诉讼需要投入的时间和金钱，以及成功的可能性，最后一咬牙：“算了！”我自费修车，开了一年后又将车折价卖了，之后添了不少钱换了台新车。这件事里里外外花费不少，但现在想起来我仍然不后悔当初的决定。我意识到，在索赔这件事情的决策上，我只需考虑可变成本，不需考虑沉没成本。如果同时考虑沉没成本，那么决策就偏离了事物本身的价值。也就是说如果我对“沉没成本”过分眷恋，就会导致偏离最优解。【03】为什么你会屡屡掉入成本的黑洞把沉没成本的概念延伸到职场，如果一份工作已经确定不是你想要的，你该怎么办？最佳答案是：放弃。但我们身边到处都是不愿意放弃的人。有个亲戚小刘，在一家工厂做核算员，她负责车间的工时统计和工资计算。这样的工作她做了4年，期间工资有一定涨幅，领导也许诺会提拔她做核算小组组长。当初小刘是按财务人员招聘进来的，但是工作4年，她根本接触不到财务工作，除了车间的核算工作，她没有别的技能。小刘想辞职，又心疼自己4年来的稳定工资和熟悉的环境，不走又觉得继续呆在这样的岗位上浪费时间。2016年，我出国之前见过她一面。她说面试过出纳和助理会计的职位，但要从头做起，觉得心有不甘。一晃，三年多过去了。前段时间，从她家人处了解到，她还在原来的工厂上班，现在是核算小组组长。她的工作内容和以前差别不大，只是要带几个新人。小刘明知道自己的岗位没有太大发展，却舍不得迈出一步。而现在，她更不愿意放弃自己7年的投入去寻找新的出路了，因为放弃的成本更大了。很多时候，人们本该努力排除干扰，却因为陷入沉没成本的黑洞无法自拔，不自觉地过于重视它而不断投入新的成本。为什么人们总是掉入沉没成本的黑洞？这其中的道理并不难理解。1、损失厌恶诺贝尔经济学奖获得者丹尼尔·卡尼曼曾经说：人们对于损失有着天生的恐惧，为了避免损失，挽回损失而甘于冒险。人们不愿意接受之前的投入是种错误，或投入付诸东流的现实。比起重新开始，获得新机会，他们更愿意坚持原来的思维方式或固有的东西来规避眼前的损失。2、侥幸心理相较于“从零开始”的巨大压力，人们更愿意相信：“现在也不是没有出路，我再坚持一下试试看，也许未来会更好。”3、害怕惩罚承认之前的投入是沉没成本，这意味着当初的选择毫无价值或价值不大。人们不容易接受这样的否定。对于管理层而言，这往往意味着项目或管理的失败，他们要为失败接受惩罚。这也导致了人们宁愿将错误的事情延续下去，也要极力规避当下的惩罚。【04】当损失不可避免时怎样做出明智的选择管理学大师彼得·德鲁克说：人生最悲哀的，莫过于用最高效的方式去做错误的事情。面对沉没成本的黑洞，我们到底该怎么办呢？最简单有效的方式，就是及时止损。证券交易中有一个技术法则叫：“鳄鱼法则”。它的意思是：假定一只鳄鱼咬住你的脚，如果你用手去试图挣脱你的脚，鳄鱼便会同时咬住你的脚与手。你愈挣扎，就被咬住得越多。所以，你唯一的办法就是牺牲一只脚。当损失不可避免时，放弃局部利益而保全整体利益是最明智的选择。正所谓“两害相权取其轻，两利相权取其重”，趋利避害，这也是取舍的本质。最实用的止损方法是“切换视角”。也就说一旦意识到自己掉进了沉没成本的黑洞，先跳脱既有的视角，问自己：“如果是别人面对这个问题，他会怎么做？”还记得英特尔放弃存储器业务的故事吗？上世纪八十年代，面对来自日本存储器的价格竞争，英特尔已经连续6个季度亏损。总裁格鲁夫下定决心要放弃存储器业务，然而这是一件难以启齿的事情，在所有人心目中，英特尔等于存储器。我们怎么可以放弃自己的身份？如果没有了存储器业务，我们还称得上是一家公司吗？1985年的一天，格鲁夫在办公室里与董事长兼首席执行官摩尔谈论公司的处境。突然，格鲁夫问摩尔：“如果我们都下台，另选一位新总裁，你认为他会采取什么行动？”摩尔想了想，说道：“他会放弃存储器的生意。”之后，格鲁夫果断放弃了存储器业务，最终把英特尔转变成了一家微处理器公司，英特尔重现生机。当我们懂得切换一个视角或身份，就会从当前的思维困境中抽离出来，重新评估事件，做出正确的决策。止损，是对自己过去的付出和努力做出新的评估，而不是全盘否定，它是你重新复盘后对未来的反向确认。余生不长，在你的生活里，让过去的过去，让未来的到来！（图片来自网络，版权归原作者所有）遇刺18天后，任达华拒绝赔偿：比坚持更难的，是懂得及时止损 刘润 : 《刘润：真正的利他，是一种在细微处都能开花的修养》一个人的境界，可以从他的言语中感受出来。提问：你一位朋友，在微信上激动地对你说，&quot;你知道吗，美元对人民币已经破7了&quot;。而这则消息，你昨天就知道了。你会怎么回答？下面的三种回答，代表了三种境界。1）我早知道了。这么回答时，你内心的BGM大概是这样的：朋友圈昨天都已经刷完屏了，你今天才知道啊？这也太晚了吧！这时，如果你愿意，你可以跳出自己，在半空中旁观自己。你会看到一个略带优越感的自己。你会忍不住脱口而出：我早知道了。这句&quot;我早知道了&quot;，背后是一种你可能藏在底层潜意识里，你的上层意识没有识别到的情绪：怎么样？我比你消息灵通吧？我比你优秀吧？这句话一说出口，你的小小的&quot;自我&quot;（ego），就闪了一下光，轻轻地刺了一下你朋友的眼睛，刺了一下他的好意。2）确实如此。这句话的境界，就比“我早知道了”高一个档次。为什么？因为如果你站在半空，你会看到顺着这句话飘到你朋友那里的信息流，不是“我比你聪明”，而是“你是对的”。不要小看这句话。能说出这句话，说明你已经挣脱了小小的ego的束缚，能够把心思放在别人身上。所谓的教养，就是让别人觉得舒服。如果你被ego束缚，强行让别人舒服的代价，就是自己不舒服。你会觉得：他不知道我比他早知道，这太亏了。我要让他知道。算了算了。为了他舒服，我忍住不说吧。这不是教养。这是假装的教养。你让他舒服后，你也很开心，这才是教养。能做到这点的前提是，你已经挣脱了自己的ego。这就是为什么我说，能发自内心说出这句话，段位高一个档次。但，这不是最高级的。3）收到，谢谢。最高级的表达，是说：收到，谢谢。为什么？因为这时，你关心的已经不是他说的话本身，而是他说这话背后的动机，和情绪。他为什么要告诉你“汇率破7”了？因为他关心你。他觉得你有必要知道。他怕你不知道。你如果把心思放在他的动机和情绪上，就会满怀感激。这时，你是不是更早知道，还重要吗？一点都不重要。表现自己聪明重要嘛？一点都不重要。还很傻。你应该由衷地谢谢他。你应该鼓励他。这样他才会在下一次遇到类似情况，会想起来，你应该知道，会想起来告诉你，而不用担心：会不会他已经知道了？所以你怎么回答？我知道了，但还是谢谢你。这么表达进步很大，但还是会让他有些沮丧。你要说：收到，谢谢。你不用说：要不是你告诉我，我还不知道呢。你不用这么说。这么说是撒谎。但你可以忽略你已经知道这个情节。因为它不重要。“收到，谢谢”是情绪，是鼓励，是事实。最后的话：今天为什么要写这篇文字？很多人说，“利他，是最伟大的商业模式”。但我想说，真正的利他，不仅仅是我带你分钱。真正的利他，源自一种根植心中的价值观，一种在细微处都能开花的修养。你的员工，你的合作伙伴，你的客户，不知道为什么，就是很喜欢和你在一起。你也不知道，为什么你就有一种吸引力。这种吸引力不是装出来的，而是突破了自己那点ego之后的修养。2019-08-06点赞数:133评论区：深爱巴蒂 : 档次不一样 Answer : 收到，谢谢。 旅行君主✅ : 好的 再见陛下 : 我会说：啊是嘛(≧∇≦*)，我刚也听说了，咋回事呀。这是啥心理呀 楼楼楼 : 学习了 张勇|健康营养|杭州 : 确实，很多时候跟事实真相相比，我们所说的话会对方情感和心理带来什么样的影响更重要 武恒标 : 收到，谢谢 工 : 狠斗EGO一闪念到这个地步，我的天! 谢谢润总，我知道了[微笑]

刘润 : 昨天，我们在一个群里讨论老人带孩子的问题。有群友认为，之所以不让老人带孩子，是因为我们父母这一代人，其实没有真的&quot;带&quot;过孩子，没有经验。老了后，不可能立刻成为专家。有群友认为，父母，和爷奶关于带孩子的争夺，其实是权力之争。谁带孩子，谁就掌握了下一代。比如过去某个朝代，皇妃生子要杀掉皇妃。因为怕她们权力过大。 这些观点，我觉得都挺有道理的。但我认为，我们和爷爷奶奶关于教育理念的不同，也许还有别的原因。我们30岁生孩子，父母60岁。一个是而立之年，一个是耳顺之年。而立之年的你，最看中的东西是什么?成就，责任。你如果不思进取，就被陪朋友们耻笑，同事们排挤。你知道竞争的惨烈，你知道成就的重要。同时，成为父母的那一天，你正式长大了。你更有责任感，你知道肩负重任。你知道成为一个具有完全人格的人，有多重要。你要你的孩子，也有责任感。所以，你所有的教育导向，都指向于此。你以为，这就是正确的。但是，耳顺之年的父母呢。他们最看重的东西，是健康，是快乐。 你所经历的一生，他们都经历过。你认为的轰轰烈烈，他们都体会过。他们也曾认为成就和责任高于一切。可是，老了之后，他们开始意识到，那一切曾以为高于生命的意义，其实没有生命重要。他们开始顿悟：平平淡淡才是真。 所以，他们的教育导向，就变成了吃饱了，长高点，多睡会儿，别被人欺负，比什么都重要。你说，你们这是不对的。你们不会带孩子。 但是，你想过没有，你，不就是他们带出来的吗？你的优秀程度，就是他们带孩子的水平的优秀程度。他们不是不会带孩子，只是他们的价值观发生了变化。某某的父亲是军人，小时候对他的教育非常严厉。今天他非常感谢父亲。但是现在，父亲对孙子却百般溺爱。你都看不懂。为什么？是他突然忘了怎么带孩子了吗？并不是。而是因为他开始认为那些事情不重要了。健康，快乐才重要。 同样道理。觉得这重要、那重要的你，今天对你孩子照英文书养。但是等你老了，面对孙子时，那种一个走向尽头的生命，看着一个新生的生命的喜悦，会让你忘了书里所有的原则。所以，关于带孩子，不是老人&quot;会不会&quot;的能力问题，而是老人&quot;愿不愿&quot;的心态问题。教育，基本都是投射。你希望把而立的自己投射在孩子身上，还是把耳顺的自己投射在孩子身上？ 最后说一句。我们都会变。我们老了会觉得自己年轻时傻得可笑。但其实一点都不可笑。你再年轻一遍，还是会那样。 年轻的你呢，也会觉得你父母蠢得可爱。但其实一点都不会蠢。你老了，也会那样。这不是进步。这是轮回。周而复始，生生不息。2019-08-06点赞数:121评论区：谢鹏 : 的确如此，我自己对父母态度的转变，正是从自己有了孩子以后开始的！不养儿不知父母恩，说的太对、太好了！[拥抱] 小小朱 : 确实如此，人在不同阶段，认识与行为就不同。就如人在爬山，爬到不同位置，看到的风景也是不一样的 嘉玲 : 投射 这个词用得太精准了 jobs潘｜儿童保 : 分析的好，但是我还看重一点，很多时候老年人带孩子，不讲就卫生，我父母还好，但是很多孩子家里老人又要带孩子又不想太累，经常带孩子去各种集体儿童游乐设施玩耍，要知道各种病毒细菌在那里互相交叉感染是最厉害的，流感爆发也是最容易受感染的区域。 所以，我的孩子我可以控制的很好从来不去那些地方玩，但是别人家呢？他们去玩生了病上幼儿园也会传染到我的孩子。所以，过去两年，我一直在班级群里充当卫生监督的角色，在群里经常发儿童医院人满为患的视频，给这些家长看，然后我说：你们看看哦~~，想去排队看病的，就让老人多带孩子去游乐场所多玩玩😂 没想到效果还真不错，过去的一年时间班级里很少有小盆友流感了😂😂😂😂 郭惠 : 我的孩子一直母亲带，也有时思考过这个问题，但后来一想，母亲带比我们有经验多了，毕竟她也带出了如此优秀的我们[偷笑][偷笑]，只是有一点，自己也要多和孩子交流，多创照一些让孩子了解这个司机机会 Lucky琳|星空画* : 对啊 谁都会老 不要嫌弃自己的父母唠叨 那是他们爱的表达。或许等我老了 我也会唠叨 像爸妈一样 哈哈哈 多一些理解和包容就好了 加乘 : 这个还真不好说，毕竟30年前的那个时代和现在的时代不一样，工作和生活方式也不一样，承受的压力也不一样。等我60的时候带孩子还真不确定跟现在我的父母是不是一样。哈 荧光 : 真的是这样，每个年龄阶层看重的东西不一样，导致教育方式的差别

润总您好作为您

2019-08-06 #刘润
(提问)Joey : 润总，您好！作为您微软曾经的同事，为您今天的成就由衷的喝彩，而作为您5商的学员，很开心每天开车上下班能有您的声音相伴左右。我来澳洲10多年，最近一次创业算是最成功的一次，但是现在发生了一点问题，想咨询下您的意见！因为提问有300字的限制，我把具体合作缘由放到了图片里，附在下面，辛苦润总了！作为5商的学员，我承认我们的合作从开始就有很多问题，但是可惜那时候还没有5商的课程，也没有股份合作的经验，现在不想再去纠结过去的错误，只希望润总能基于目前我们的情况指点迷津。1. 60w是股东投资还是私人借款，如果撤资，股份如何计算2. 工资怎样核定3. 如果双方意见不统一，润总什么建议2019-08-06回答：1）看清楚价值创造者是资金，资源还是人力。谁创造价值，谁分配利益。新公司股权分配应更多地关注股东未来给公司创造的价值，而不是股东的初始出资或者创业前的成就。 2）定好结构，即设置股权比例。两个股东的股权分配原则，承担主要责任的老大要有绝对控股权，51%以上，一般来说，控股比例要超过67%，这样既是为了激励，也是为了后期决策。 3）初期她出资金，你出经营能力，形成“公司资产”。两人“共同认可”股份占比5:5均等，共享公司的所有权、处置权和收益权，这三个权利是对等的，也就意味着这60w的资金，其实是默认属于你们共有的公司资产。 4）如果想把共有资产（仓库）卖了兑现60w资金，理论上讲，你们应该是一人一半。 5）对方想要把最初的60w全部拿回去，本质上是“撤资”行为，如果要撤资，意味着自身股份出让，如果你同意兑现，其实是从她的手上买走了剩下的那一部分股权。股权比例和价格则需要协商。 6）工资和分红权要分开计算。主要股东应该拿的是“社会平均工资”，甚至再低一点。不能靠工资来获利，不然会产生各自的利益诉求。谁职位承担的责任越大，创造的价值越大，就拿更多的钱。 7）股东应该是从分红里赚钱，而不是从工资里赚钱。你们需要互相评估对方的表现以及对公司的贡献，要讨论清楚谁承担主要责任。 按职位承担的责任拿到社会平均价钱之后，公司可能还有剩余盈利，此时再按照股权结构来分润，如果股权比例是3:7，就按3:7分，不存在出现对等情况，好像工资就应该一人一半一样。这是不对的。 谁创造主要价值，谁分配主要利益，谁创造 次 要 价 值 ，谁分配次要利益，谁不创造价值，谁不分配利益。 8）对方之前的工资是你的70%左右，也是值得商议的，是否需要支付70%，要看这个职位创造的价值，如果你找社会上别人承担这个职位的责任，需要付出大概多少钱？ 以此来确定比例，不要将友情和利益纠葛在一起，工资只跟承担责任有关。 9）虽然对方初期出资60w对公司有历史资金贡献，但贡献期限较短，若以当下持股比例进行年度分红仍然存在问题。 可将年度分红核算的依据分解为历史股权与当下贡献两部分，并各自赋予权重。 比如：上年度股权比例权重为 40%，本年度业绩贡献比例权重为60%，那么，本年度分红比例 = 上年度股权比例 ×40%+ 本年度业绩贡献比例 ×60%。 把股权分配依据与公司价值链贡献因素挂钩，充分考虑合伙人在公司发展不同阶段所做贡献的差异，据此进行动态分配，分配依据的原则是：合伙人对于公司的贡献值大小。 10）合伙人贡献因素有很多，比如开发维护客户订单资源、开拓供应商资源、出资额度、品牌运营等作为贡献因素并赋予相应权重，依据合伙人的能力预估各自对每一项贡献因素的投入，然后计算初始股权分配比例。 11）公司股权是一个需要不断动态调节的过程，需要你们来敞开心扉讨论并达成共识。点赞数:66评论区：风 : 老师辛苦了 提问同学附的图片好像没看到在哪 树 : 我记得这味同学之前发过这个问题，可以搜一下。 风 : 好的，谢谢 雷子 : 润总说的很到位，我想这应该是最公正的处理方式，只是如果这种公正的处理方式会损害其中一个合伙人的直接利益，怕是再明白的道理也很难沟通好[尴尬] 工 : 第十一条，打开心扉后，前十条才能发挥作用。利益问题，永远又是一个情感问题。建议你把这十一条给合作伙伴看一下，这些专业知识，没有人讲，是不可能从脑子里长出来的。同伙人如果死也不信，不愿面对，恭喜你，早分早超生。如果和你一样恍然大悟，恭喜你：不分也超生。 Joey : 这里是图片，谢谢关注！ Joey : 终于等到润总的回复，真的非常感恩！太有价值了！ Joey : 这个评价很精辟啊，必须打赏！
