2019-08-04 #刘润
居尔提 ma0x : 最近回去翻了几本看过很多次的书，赚钱的两种策略，赚快钱与挖金矿。经济学基本原理是，任何一个行业，长期来看是没有超额利润（easy money)的。你的销售额只能覆盖你的成本，包括原材料成本，销售成本，人力成本等等。如果有经济利润，其他人就会加入这个行业，供给增多，价格下降，直到没有超额利润。也就是说，如果一门生意好赚，其他人就会一直涌进来，直到不好赚为止。除非你能用“其他方式”锁住门槛，变成特许行业，不然什么行业最终都会没有赚头。依照这个原理，赚钱只有两种方向：1.机会财：不停的寻找新的赚钱机会，在其他人来到之前卡住位置。等其他人涌入，就把锅甩给他们，自己换一个战场。我把这个在这叫“赚快钱”，短期收入。2.经营财：能把成本降的比同行更低，在同一个领域中不断的踩死竞争者。我把这个在这叫做挖金矿，长期收入。照这个原理来分析，多数人的创业，都是想赚快钱...做网拍也好，眼镜店也好，奶茶店也好，在夜市摆摊卖消夜也好，或者骑车送餐也好，其实都是靠速度赚钱。因为，这些是没有门槛的行业，长期来看净收入不可能太高。这种类型的工作，在初期的机会财阶段结束后，会有大量的同行出现。想赚后期的经营财，在不违法的前提下，能赚的都是辛苦的皮肉钱。那该怎么办？我们把赚钱方式分成短期与长期来分析：先说赚快钱，短期收入：短期收入跟长期收入不同！一是可能存在竞争不完全的市场，这个时候新市场的出现带来的需求大于供给，只要你做，就能赚钱。因为市场做出调整需要时间比如，你比外卖大军更早开始送外卖，由于当时人没那么多，你还真的赚进月薪两三万。理由并不是你有多厉害，纯粹是因为竞争很少，好赚的钱没人抢。二是存在资讯不对称，人们没有发现你做的事情能赚钱。这个必须有背景，有条件，没有的人不要妄想。比如你家里有一个特殊配方，或者你搞出一个新东西。别人要想也赚一手，必须花很多时间研究。你做这个，一定时间里面可以类似于垄断的赚钱。所以，寻求短期收入的思路有二：第一，寻找一个新兴行业，投入进去做。第二，如果自己有什么别人没有的资源，想办法用来赚钱。但很容易可以看出来，这两点都不简单。投入新兴行业需要钱和一定的资源，发现新市场也需要成本。至于自己想搞出别人没有的东西，更是困难。而且，短期市场最大的问题是，不确定未来会怎么样，从某种程度上说，投入短期市场的人，最后更多积累的是做生意的经验，想长久做下去，没有保障。再来是挖金矿：长期来看，想赚钱必须要提高个人的价值。个人的价值不一定是说技能高低，比如你学某种技术做到行业第一，这是一个方向。你掌握的资源量也是你的价值，比如你要做手机，你得首先能找得到有关的供应商，设计师，程式设计师。如果你能找得到，相比于那些找不到的人，你的价值就高多了，这其实就是人脉、通路。总结一下，如果你现在一无所有，想要赚大钱，长期来看没有捷径，两个思路。1.好好读书，学习真正有用的技能，吃技术饭，多年钻研成宗师。2.累积资源和人脉，进一个行业把各相关的产业链都做熟做好，相对于那些没有这些经验的人，这就是你的收入。千万不要想著有什么捷径，更不要唱高调。如果你家里有钱有资源有人脉，这就是你的捷径，果断运用。如果你家里没有什么可以支持的，多读书，然后找一个正职工作好好干，累积资源再创业，可能最有效。#end按赞打赏是持续输出价值的动力，写这样一篇帖要花一定的时间精力，我在这发的帖都是首发或只发在这，大家去看各个星球的状况，我这样做显然不太一样，单纯分享有价值内容。2019-08-04点赞数:129评论区：K : 请问一下，那几本看过很多次的书是什么书？谢谢。 刘润 : 写得非常好。用熊彼特的话说，“只有创新才能带来利润”。我把它分为：红利，工资，利润。以后写文章与大家分享。打赏了。 居尔提 ma0x : 谢谢刘老师 刘润 : 给你私信了。求加微信。 居尔提 ma0x : 收到，谢谢老师 居尔提 ma0x : 1. 食利者政治经济学（中文译名） 2. 利润率的政治经济学 小张 : 有点像 受阻模型组 飞越疯人院 : 要不自己是金子，要么就是能发现金子

润总你好！公司做的是新生儿的诞生礼(喜蛋喜饼)主要天猫零售，线下喜铺经销。因经营不善从月销300万跌至30万，欠债几百万，日常开销捉襟见肘，欠薪老员工。老板不放弃，但经营上很多问题1)流量资金都缺乏的情况下，应求差异化出小众礼盒，还是先满足大众需求？2)特殊人群和时期一次性购买商品，没有冲动消费和复购，在抖音等平台上销售，合适吗？(没钱请网红，自己做但不擅长)3)同行刷单做数据，抢占流量形成马太效应，我们纯自然生长，如何应对竞争？4)口感品质业内是公认的，但价格高，批发量起不来，品质价格如何平衡？5)为了线上性价比竞争力，定价过低就会影响批发业务，两者该如何平衡？望解感恩！2019-08-02回答：1）差异化小众礼盒和满足大众需求，两条路径都是正确的，没有孰优孰劣之分。这背后对应的是企业不同属性的能力。 所谓的战略，就是目标和能力的匹配。 你决定做大众化产品，对应的能力可能是能做到极致的性价比，用巨大的规模效应带来价格成本优势。 你决定做差异化小众产品，对应的能力可能是对用户需求和痛点把握的非常精准。 你想选择哪条路，要看你擅长哪种能力。 流量资金都缺乏的情况下，应该追求的是活下去。 找到一个独特的销售主张，向你的客群表达一个竞争对手做不到或无法向消费者提供的主张，以此打动吸引消费者购买。 而这，可能需要你从中国上下五千年的传统文化之中探寻答案，找到刚需，用引流品和利润品获得利润。 比如：买新生儿诞生礼（喜蛋喜饼），附赠长命富贵锁，开过光的幼儿压惊辟邪的独家产品等。 2）新生儿诞生礼，是一个非常明确的需求，受众在特定时期会自然而然的上网搜索，可以做些百度关键词优化，占据流量触点。 特殊人群和特定时期一次性购买商品没有复购。要看你站在什么角度来看待这个问题，受众如果自己不复购，但是如果产品超预期，会忍不住在亲朋好友间推荐，变成口碑分享，形成源源不断的客源。 抖音作为新兴的流量入口，有一定的转化率优势，但不是所有的品类转化率都很好，在目前你资金不足的情况下，先活下去。 3）一定有不靠刷单做的好的产品，不能简单归因于别人做的好的都是因为刷单，就算流量受限，你还可以在转化率、客单价和复购率等环节下足功夫。 研究如何联合线上和线下多渠道融合营销，用更好的品质，更优惠的价格，更舒适的服务体验来增强消费者满意度。 互联网电商只是一个工具，并没有改变经济学原理，并没有改变商业的本质，只是提高了商业的效率。 不要把线上店当做完全依赖的单独渠道，而应是经营的一个子模块，线下婴童连锁店、KA卖场、大型百货商场及个体婴童店等业态都应该去尝试合作。 4）你手上有一颗特别甜的梨，而消费者想要的却是苹果。 你需要找准购买欲，受众买喜蛋，核心需求真的是为了吃吗？ 更多的原因可能是仪式感和讨吉祥喜气。产品品质要好到超预期，忍不住分享，比如在定制化，文案，包装设计，产品原料上，有独特的价值，消费者才会为价值溢价付费。 5）不要试图满足所有的消费者。继续保持产品的高品质，同时推出中低价位的产品系列，通过对价格策略调整，对目标消费者市场进行细分，考虑清楚是走高端路线还是低价销量路线，这是完全不同的打法。 如果你是要“救命”，那么你现在可能暂时是管不了线下和线上差价的批发问题的，而是应该走量，以量换取利润，把正向的现金流循环体系先搭建起来。点赞数:69评论区：jobs潘｜儿童保 : 我加入微软时曾被

2019-08-02 #刘润
刘润 : 我加入微软时，曾被一句话深深激励：我们的产品，改变着上亿人每天的生活。是的。这句话反过来讲就是：上亿人的生活，就在产品经理们的一念之间。给你讲个小故事。今天发生的故事。我今天出差，从上海去杭州。因为一些原因，到高铁站早了。于是想，能不能改早一班，提前到杭州，入住酒店。时间花在喧闹的高铁站，不如花在安静的酒店。还能干点活。于是我去排队。虹桥站内有“售票”，“改签”和“退票”三种窗口。我拿着票，排进了“改签”的队伍。终于排到我了。我说请帮我改最早的去杭州的高铁。改签员看了一下屏幕，说：你的票已经改过一次了。不能再改了。是的。我的行程一直变化中，助理已经帮我改过一次行程了。我说：那怎么办？她说：只能退了再买。我说那就帮我退了再买吧。退票费我付给你。改签员说：我们只改签，不退票。退票在另一个队。好的。一年出差100多天，对这样的“游戏设定”，我已经被磨练的毫无怨言。没搞清楚状况，是我的错。我的错，我的错。我赶快排入了“退票”的队伍。终于排到我了。我说请帮我先退票，再买最早去杭州的高铁。退票员说：我们只退票，不卖票。你退不退？我说：啊？那你先帮我看看，前面班次还有座位吗？有座位我就退。免得退了才发现没票。退票员说：我们看不到。只负责退票。你要查有没有票，要去售票窗口查。我问：那我查完能直接买吗？退票员说：不能。要退掉才能买。有票在手，是不能再买附近时间的票的。我的头“嗡”地一声。这不就死循环了嘛？我去排“售票”的队，查到前面班次有票，但不能买。回来退票。退完后再排队去买票。那时候没票了怎么办？没办法。只能这样。这是谁设计的？产品经理。我能理解，产品经理的每一个设定，都是有道理的。比如，有票不能再买，是为了防止一个人买好多票，真正想走的人没票。只能改签一次，是为了车票安排的稳定性，最优化全网行程。等等。但是，这些放在一起的复杂性，让产品经理忽视了“二次改签”的悖论。产品经理办公室一念之间的疏忽，让我在高铁站站在三个窗口前进退两难。但是，活人怎么能被尿憋死。我突然想起来，自动取票机上，也能查询余票啊，还不用排队。我立刻冲到自动取票机前。刷身份证一查，果然前面一班还有两张商务座。从自动售票机转身回去退票，再回到自动售票机上买票，大约要5分钟。这时，我要做个决定，要不要冒一个险：退完票的5分钟内，这两张商务座正好都被卖完。应该不会吧？要卖完早卖完了吧？不会等到这5分钟吧？我决定冒这个险。我冲回最短的退票窗口，退了票。退票员收了我20%退票费。她还要撕发票给我。我不要发票，只想她把身份证尽快还给我。拿到身份证后，我立刻冲到自动取票机前，非常熟练地开始买票 刘润 : 昨天发了一篇文章，讲到阿德勒的“课题分离”理论，说判断一件事是谁的错的标准，是：谁的损失大，就是谁的错。什么意思？比如，有人在地铁里踩了我一脚。谁的错？我的错。啊？明明是他踩了我，为什么是我的错呢？难道我不应该要求他道歉吗？你可以要求他道歉。但是，道歉有什么用？而且，你要求他道歉，不需要花时间嘛？他耍无赖和你吵起来，不更需要花时间嘛？你的时间没地方花了吗？这时，对方可能反咬一口：你怎么把脚乱放啊？！你怎么办？你要说：我的错，我的错。然后，心平气和地走到旁边。因为，你的时间，比他值钱。这就是“谁的损失大，就是谁的错”。但可能还是有同学不完全明白。今天多讲两句。一个人心中，应该有三种“对错观”：1）法学家的对错观；2）经济学家的对错观；3）商人的对错观。什么意思？我一个个来讲。1）法学家的对错观。我举个例子。一个坏人A，诱骗好人B到没有锁门的工地C，然后B失足摔死了。请问，这是谁的错？这当然是A的错，这就是蓄意谋杀。这还有什么好讨论的。是的。如果证据确凿，从法学家眼中看，这就是A的错。但是，这种大快人心的对错观，不一定能防止类似案件下次发生。法学家做不到的事情，经济学家也许能做到。2）经济学家的对错观。经济学家有不同看法。B被坏人A，诱骗到没有锁门的工地C摔死了，是C的错。啊？为什么啊？这也太冤了吧？这是因为，整个社会，为了避免B被A诱骗到工地的成本，比工地C把门锁上的成本，要高得多。如果惩罚了工地，虽然工地觉得冤，但是以后工地就都会把门锁上了。这样的事情，会大量减少。所以，经济学家，是从“社会总成本”的角度，来判断一件事的对错在谁。虽然听上去不合理，但有时比纯粹的“道义”更有效果。3）商人的对错观。但不管是A的错，还是C的错，B都死了。把谁抓起来，B都无法起死回生。所以，从个体利益最大化的角度，B只能怪自己。我的错，我不该蠢到被A欺骗。只有这样的认知，才能保护B自己。一个人走在人行横道上，一辆卡车呼啸而来。所有人都大声呼喊他让开。他淡定地说：他不能撞我。他撞我是违反交规的，他全责。我就不让。最后这个行人被撞死了。这是谁的错，卡车司机的错？当然。但是这样的认知无法救命。这个行人要认识到：这是我的错。我应该让开。因为死的是我。你看法官认为A错，经济学家认为C错，商人认为B错。这就是一个人心中的三种对错观。如果你是评论家，你可以选择法官的立场；如果你是政策制定者，你应该选择经济学家的立场；但如果将要失足摔死的就是你自己，我建议你选择商人的立场。我的错，我的错，都是我的错。因为我损失最大。谁损失大，就是谁的错。2019-07-30点赞数:119评论区：福州陈巍 : 不同角度切入点不一样 旅行君主✅ : 明白了，有道理 海东 : 人生观，价值观，世界观，还有对错观，消费观，好多观啊 李崧 Stanley : 火车疾驰而过引起的火星燃烧了铁轨旁边晾晒的棉花，谁的错？经济学家说是棉农的错，因为他避免损失发生而采取的成本，比整个社会为避免之而采取的总成本，要少得多。从商人的角度，还是棉农的错，可预见后果发生时谁的损失大就该怪谁。 有脾气 : 这观点够味儿，确实如此，利益相关度决定立场，每个人描述的都是自己眼中的世界。 焦浩 : 昨天看《哈佛商业评论》里有一期专门讲时间的文章《幸福的时间》写的很不错。欢迎大家查阅。 工 : 不服不行，一个小题目，挤压出一个大道理。学到一点思考方法：一个问题如果想不清楚，一定是因为问题本身分的不够细。 自强不息 : 副驾驶的乘客，总是说“干嘛让别人插车，撞上去，他全责，他全责” 我淡定刹车：他全责又怎样，陪他报保险的是我，去修车的是我，找他理论赔偿的还是我，是我，是我，还是我。

咨询顾问的主要工作

2019-07-29 #刘润
刘润 : 咨询顾问的主要工作之一，是给企业提建议。建议，必须针对问题；问题不同，建议也不同。我把建议，大体分为三种：救命建议，治病建议，和养生建议。1）救命建议。突然，一位同事晕倒在你身边？你应该怎么办？立刻拨打120，送到最近医院。这时候，你会说等等，我查一下，哪家医院处理这种突然晕倒类问题最擅长吗？你会看看最牛的主任医师哪天上班吗？你不会。因为，这时候“治病”不是最重要的，你第一要务是救命。同样道理，一家公司现金流断裂，你给他的建议应该是“提炼愿景、使命、价值观”吗？不。这时，你应该卖掉或者关闭不赚钱的业务，甚至裁员，阻止大动脉失血。可是，关闭业务、解雇裁员，会有副作用吧？IBM传奇总裁郭士纳接掌帅印后，裁掉几万名员工。被裁掉的人，一定是怨声载道；留下的人，一定是士气大落。但是，连命都快没有了，救命第一。救命，大于治病。2）治病建议。到了医院，命保住了。这时，你问医生怎么回事。医生说，他晕倒是因为一种罕见心脏病。然后，你开始和医生讨论这种病的治疗方案。是搭桥，是移植，还是装人工心脏。如果是装人工心脏，哪家医院手术的成功率最高。这时候，你会对这个病人说“少油少盐多健身，就没事了”吗？你不会。因为养生习惯、保健食品，也许可以防病，但是不能治病。病一旦发生，只能积极医治。同样道理，公司产品质量出现严重问题，导致大量客户投诉、退货。你给他的建议应该是“全体员工加强质量第一的文化建设”吗？不。这时，你应该召回产品，弥补客户损失。可是，召回产品，也是有成本的吧？当然。但如果不这么做，公司声誉将会大损，病会越来越严重，最后要你的命。临时抱佛脚讲文化，治不了病。治病，大于养生。3）养生建议。你的朋友出院了，问医生有什么建议。医生说，饭后百步走，能活九十九。合理饮食、无烟无酒、心情愉快、适当运动。这些都是养生建议。没有命悬一线、病入膏肓的人，感受不到这些建议的价值。同样道理，公司在“健康”时，我们要关注客户，投入研发，公平激励，建设文化。这些听上去很虚，什么效果都没有。但却可以治未病。一个优秀的商业顾问，会冷静清晰地知道，什么时候给什么建议。救命时，冷静地壮士断腕；治病时，咬着牙刮骨疗伤；养生时，坚持要跑步爬山。而一个糟糕的商业顾问，在顾客命悬一线时，建议你治病；在病入膏肓时，劝解你养生；在一切正常时，建议你饮鸩止渴。而你，必须具备清醒地知道自己处于哪个状态的能力。2019-07-29点赞数:95评论区：slj : 药💊，工具，解决方案都在库里，但何时用，量多少，怎么用，才是最难的抉择的。想想都头疼！ 邱斌 : 怪不得都要看情况呐 方一茗 : 救命是没做重要紧急的事 治病是没做重要不紧急的事 养生是做了上面两个事，但做的不透不持续 刘萍 : 浅显易懂，大道至简。 Vivian : 含金量太高，我要好好地琢磨消化。 腾飞盛业-ALEX : 养生之道何其多，找到正确的，坚持不懈做，很难[呲牙] 王天海 : 发现问题，提出问题，分析问题，解决问题。以问题为导向，以问题为核心 Teh 郑 : 简单易懂，背后功夫少不了
