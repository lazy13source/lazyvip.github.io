1、关于复购率的理解：装修与建材、家具等属于低频消费，平均装修频次为8-10年一次，故复购率不仅仅应包含老客户复购率，也应包含老介新率（老客户转介绍新客户）。 2、提高复购率及老介新率的方法，最不重要的就是营销。在耐用品、低频消费的家居建材领域，大部分老客户不会仅仅因为你给老客户5-10%的优惠，就复购，更不会因为你给他5-10%的返利，就推荐自己的亲戚朋友同事邻居买【注：超过10%，大多数品牌在目前促销最低价格基础上，也很难给的出】。复购的逻辑是，我认你；推荐的逻辑是，我相信这个品牌值得我用自己的信用做背书去推荐。而不是任何营销的手段。 3、提高复购率及老介新率的根本思路，是老客户在交易过程中或者交易完成后成为品牌的铁粉。 4、举例： （1）欧派某城市2018年销售额13亿多，其中高达60%为老客户转介绍、老介新，核心是追求客户高满意度，全流程、全员考核用户高满意度。 （2）曲美家居某城市经销商，同一精装房楼盘，交付3个月内，成交率占入住率50%以上，并非是营销原因、价格原因，核心是在超越同行的设计方案、安装服务基础上，又增加让业主特别省时省心的第二管家服务，包括为业主看管工地，拿送钥匙，开窗通风，监督灯饰软装电器等非曲美产品安装等免费服务。 （3）曲美家居某城市经销商，今年开始拓展整装全案服务，大到户型布局，基础装修，小到一副挂画，全流程配套服务，效果图真正的所见即所得，收费无过程增项，服务省时省心省力，从商业模式上让客户享受到了目前市场上最极致的服务，几乎所有服务过的整装客户，都成为曲美整装的铁粉。 张红梅LEO！YES* : 5、成为铁粉的核心思路：以终为始，尽可能在自己所在的商业模式下，创造全流程高满意度 6、提高全流程【或者是客户全生命周期】高满意度，让客户成为铁粉的具体举措： （1）品质、交期是基础，但这是主要受厂家影响的要素，暂且不谈。 （2）提高前期接待环节满意度：门店接待、社群互动、样板间接待高满意度，如：考核销售顾问、设计师服务满意度、专业度 （3）提高设计方案高满意度：如：考核设计师服务及时度、服务态度、方案满意度 （4）提高施工、安装过程高满意度：装修公司，这个环节要特别注意。定制类产品，包括木门、门窗、定制家具、橱柜等考核安装一次完成率 （5）提高售后问题响应及时度及落地满意度：不说了，大家都懂。 （6）是否有与购买产品相关事项配套服务及满意度：比如欧派某城市，为客户提供旧厨房拆除、垃圾运输服务，且收费明确；而同一市场上的竞争品牌，一家不提供垃圾运输，一家不确定收费要师傅到了现场再确定。这不仅阻碍了成交，即便成交了，也极其影响用户的体验和满意度。 （7）是否有产品交付后的老客户增值服务：比如欧派的金保姆服务，每年为老客户免费清洗厨房、橱柜、烟机灶具；慕思床垫每年为老客户免费除螨虫。 （8）是否有老客户社群服务：针对高端客户，比如VIP老客户针对不同兴趣爱好，定期组织纯粹的不牵扯营销活动的线下互动，如红酒品鉴会，书法品鉴会。 7、提高全流程高满意度，让客户成为铁粉的保障：内部构建文化、架构、考核、机制保障；外部，整合资源解决客户难题。 （1）首先老板要有把提高全流程服务满意度，作为头号工作来抓的意识 （2）全员形成为以追求用户高满意度的文化 （3）设计相关岗位：根据规模，选择全职、兼职。比如设置设计主管（总监）岗位，以培养设计师，审核设计方案，陪同谈单，提高设计方案满意度；设置施工监理岗位保障施工进度、提高施工满意度；设置金保姆岗位，做售后增值服务（比如清洗产品、实木产品保养等）；设置专职（规模小兼职）售后专员，第一时间响应售后问题；设置客服专员，对每个客户进行服务全流程回访考核等等。 （4）设计考核及相关考核标准：全流程制定标准动作，以表格形式呈现，请客户填写满意度考评，或者设置客户与绩效挂钩。 （5）整合资源，为客户提供配套增值服务，即便收费，也解决了客户的难题，既提高了成交率，也提高了满意度。 张红梅LEO！YES* : 你好润总我司有5

2019-08-20 #刘润
(提问)BN拳击品牌-源头工* : 你好，润总我司有5个销售人员，每一个业务员的工作年限和客户积累不一样，所以给他们设定的月任务量也是不一样，设定120万是他们必须需要完成的计划目标，做到120万他们没有奖金，如果他们5个人达到140万了，看谁超额部分越多谁就是第一名，第一名奖励6000元，第二名奖励4000元，但是有个问题就是对销售任务目标高的人不公平，因为他的销售基数大，超额部分就比较小，虽然他总销售额是第一但是他超额部分不是第一还是拿不到奖金。请问润总关于这个公平性问题如何去设定？2019-08-20回答：1)每个企业都会有其生命周期，从创立期、快速发展期、步入成熟期、再到衰退期，企业应根据每个生命周期阶段发展现状和发展目标制订不同的销售目标、销售策略并且采用与之相配套的薪酬模式，促成销售目标的完成。 对销售人员来说，不能因为年限积累不同，就区别对待。 不同之处应该体现在工资和奖金的基数是要不一样的。 比如说工作经验丰富，有能力去接待更多客户的销售人员，他的工资基数就应该更高一些。 假设5个人中，A销售工资基数是10000元。 B销售能力更强，工资基数是15000元。 两者的不同之处，在于拿15000元的销售人员，身上背负了更大的销售指标配额。 销售指标的公平性，会跟他的收入公平、能力相关，而不跟他的工作年限相关。 2）每个销售人员，都应该有一套和自己工资相关的奖金包，并根据销售配额指标设计超额提成计算表。 比如A销售月固定薪酬1万元，配比5000元奖金包，50%就是5000块钱。 固定薪酬15000元的B销售，50%，就是7500元的奖金。 所以，奖金应该根据工资，是一个固定的比例关系。 他的奖金只能跟自己相关，不应该是在五个人中间去竞争，因为每个人的技术和能力不同，当然会有不公平。 这个不公平解决的根本办法是奖金应该“各自跟各自”的人，跟各自的指标相关。点赞数:70评论区：海东 : 学习了 prettyboy : 太赞了👍 站好、认识下 : 绩效奖金[呲牙] 姜义成 : 学习也 海东 : 这篇我要收藏下，然后整理成导图。这个问题，我这里也有很多商家有痛点，一直找不到专业的指导。谢谢润总， 鱼本渔 : 很棒 孔雀翎 : 没有更赞 BN拳击品牌-源头工* : 学习了谢谢

孙瑞希：生涯咨询师，个人成长教练，今日头条签约作者，青云计划外部评审团成员。专注分享成长干货，推动你的职场进化。生活中有许多人忍不住靠优越感将自己锚定在某一个阶层，在他们看来，这个阶层以下的人都很“Low”,自己就是比别人高贵。然而，真正高贵的生活，是不需要用优越感来装饰的。太有优越感的人，往往是精神上的乞丐。著名主持人孟非曾经说过一句话：“所有的优越感都不是来自容貌、身材、知识、家族、财富、地位、成就和权力，它只来自缺见识和缺悲悯。”而你总要为自己的缺见识和缺悲悯买单，或早或晚而已。【01】​你的优越感仅仅来自于没有温度的内心我不喜欢太有优越感的人。与他们相处，前一刻是天使，后一刻可能是魔鬼。他们双重标准玩弄是非，背后藏着人性之恶。我有个亲属，是某高校的教授，一直以来，我都很敬重她的学识，直到有一年见识了她通过欺负他人来垫高自己，才看清了她的嘴脸。那年春节，他们全家回东北过节。正月初三，一群亲属在一起聚餐。等待上菜期间，教授热情洋溢地回复同事们发来的祝福微信。我看见她眼角挂着笑，声音又温和又从容。春节期间留守的服务员不多，菜上得有点慢，教授很不高兴，阴沉着脸催了几次菜之后，勃然大怒。她把服务员叫到包间质问，那是个二十出头的姑娘，被她吓得一脸惶恐。从菜品质量到服务质量，从服务行业从业人员素质到消费者权益，她引经据典，上纲上线，服务员只能不住地道歉。但她仍然不依不饶，直到大堂经理出面道歉，赠送了两道菜才算了事。大过年的值班，本来就不容易，还被客人刁难，服务员委屈得直抹眼泪。她恶狠狠地瞪了一眼：“哭什么啊？我冤枉你了还是委屈你了？餐具这么脏，擦一遍再端上来！”一个电话打进来，她眉目含笑：“过年好啊，X院长……”我的心猛地一沉，对她的敬仰荡然无存，媚上欺下，不可交也！几年后，我们之间共同的亲属去世了，我跟父母商议，这种人，此生不必往来。现在的人们都喜欢谈“修养”二字，其实一个人真正的修养不在于学问，而在于有没有一颗有温度的内心。这个温度，不是表现在对权贵的恭谨上，而是看他是否能尊重和体恤弱小。真正有修养的人，不会通过欺负他人炫耀自己的身份，他们平等地对待强者和弱者，用一颗宽容的心来平视普罗大众。【02】但凡能活得让人喜欢就别活得让人嫉妒这些年，我见过了很多秀“优越感”的事，我有时候也会不经意地流露出优越感，为此我经常反省。我发现优越感通常会在两种情境下产生：一是想炫耀某种东西，不然憋得慌；二是通过踩别人来抬高自己。著名心理学家阿尔弗雷德·阿德勒在《这样和世界相处》一书中提到：我们与世界相处，免不了会遇到各种各样的苦难，面对苦难人会有不一样的应对方式：主要是对优越的渴求和对自卑的抗拒。自卑和追求优越是成对出现的。用一句通俗的话来说就是：一个人越炫耀什么，内心就越缺少什么。有，不代表不缺，小时候没吃饱，一辈子都在意。当人们受到挫折产生自卑的情绪之后，可以通过转化为动力实现补偿。这种补偿心理有时是好的，比如一些残疾人身残志坚，通过其他长处弥补自身生理缺陷，取得成功。但补偿心理也有跑偏的时候，比如过度补偿。通过炫耀性消费或建构优越感来掩盖自己内心的自卑感。有次，演员田小姐上一档节目，她先是怼了在场的所有主持人，然后展开自己的话题：“但凡能活得让人嫉妒，就别活得让人同情。”高晓松 直接纠正她：“但凡能活得让人喜欢，就别活得让人嫉妒。”阿德勒说：这种争取优越感的动作总是朝向生活中无用的一面，真正的问题却被遮掩起来或避而不谈。事实上，真正的自在和内心丰盈，是扔掉优越感，建立自信。【03】怎样建立自信让愚蠢和无知走远自卑感其实与我们童年的生活经历和父母对我们的态度有直接的关系。比如，我的那位教授亲戚，童年家境极为贫寒，经常受同学欺负，她成年之后的强悍不过是对自卑感的掩饰。一些童年被父母忽视或冷漠对待的孩子也容易产生自卑感，他们往往认为自己是没有价值的，不值得被爱的。自卑感带来的大部分怪异行为，归根到底都是因为缺爱。时过境迁，童年的事情没办法改变，但人都有追求优越感的本能。阿德勒认为，人追求优越感有两种，一种是追求个人的优越，一种是与社会共生的优越。追求个人的优越，简单来说就是通过打压别人来获得优越感，通过炫耀来获取关注。比如炫富。追求与社会共生的优越才是真正的优越。再牛的个体也是社会的一部分，我们所有的成就都是建立在现有的社会资源基础上的。追求与社会共生的优越强调的是对比过去，发现现在的自己有进步，对社会有贡献。对比别人时，是对标，即发现自己与优秀的人的差距，不断进步，缩小差距。人，只有意识到与社会共生的重要性，才能通过真正的成绩去建构自己的自信。莎士比亚曾说：“一个骄傲的人，结果总是在骄傲中毁灭了自己。”真正内心丰盈的人，从不炫耀他所拥有的一切。他们收敛锋芒，去体恤普通人生活的艰辛和不易，这才是闪烁着人性光辉的大智慧。&lt;a (提问)浙江英明乐智能科技-* : 老师你好，去年10月份，我生产了一款手机支架产品，没想到一下就火过了，但因为是第一次做生产，第一次做销售，所有都是第一次...所以各方面都把控和发挥不好！因为没有多大技术含量，12月份就有同行仿制出来，同行做的久生产成本低聚到广，我们一直主打品质和包装，而且产品实在太火，市场虽然有几家仿但是一直都是供不应求！今年3月全国仿制的不低于1-200家，所以我们优势也慢慢消耗没2019-08-14回答：1）这个世界上，你赚的钱分三种：红利，社会工资，和创新利润。 2）一个没有技术含量，没有专利的创意产品，能在短时间内赚到的钱，叫做红利。 3）红利，是短暂的供需失衡。一定会随着竞争者的加入，或快或慢地消失。 4）对此，不要任何&quot;愤恨&quot;。这是经济规律。技术含量高的，这个过程会慢一些。技术含量低的，这个过程会快一些。有专利的（并且在专利保护力度强的市场），这个过程会慢一些，否则这个过程会快一些。 5）一旦红利消失，这个行业，就开始赚微薄的利润。少了你就不干了。多了就会被竞争打薄。这部分难受的利润，就叫&quot;社会工资&quot;。 6）很多人以为自己在创业，但他其实只是在为社会打工。领着一份不受他控制的微薄工资。 7）那什么是利润呢？你必须有创新的办法，能降低成本，别人还做不到，那部分才是你的利润。 8）你的竞争对手，如果在同品质的情况下，做到比你便宜，那他赚的才是真正的利润。 9）你这个产品，真正比的不是&quot;正品&quot;（突然间的一个好创意）。你这个产品，比的是运营能力。 10）除非，你能做出一个，别人很难抄袭，或者要花很长时间才能抄袭的产品。点赞数:88评论区：展望未来🤓 : 老师您在开罗，还有时间回复这位学员的问题啊。[呲牙] 小楼一夜听春雨 : 除非，你能做出一个，别人很难抄袭，或者要花很长时间才能抄袭的产品。多痛的领悟[捂脸] ldw : 长城防御系统了解一下。把润总上面说的做到极致了 展望未来🤓 : 我想很多微小的企业都是在挣社会的平均工资。 浙江英明乐智能科技-* : 谢谢老师点评！收益匪浅！谢谢大家！ 工 : 除非，你能做出一个，别人很难抄袭，或者要花很长时间才能抄袭的产品。多痛的领悟[捂脸] 工 : 楼主能有一个好创意的能力，希望修练成一个能够稳定输出的能力 鱼本渔 : 边拿社会工资边创新
