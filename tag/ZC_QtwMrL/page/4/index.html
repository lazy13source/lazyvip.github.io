真正的高手都懂

2019-07-24 #刘润
孙瑞希-生涯咨询师 : 【真正的高手，都懂得“最大可承受成本”的重要性 - 今日头条】作者 | 孙瑞希：生涯咨询师，个人成长教练，今日头条签约作者，青云计划外部评审团成员。专注分享成长干货，推动你的职场进化。我一直在琢磨一个词：最大可承受成本。我发现无论是工作还是生活，如果你能搞清楚自己的最大可承受成本，就没有那么多纠结，你就有极大可能换来预期的进步和收获。“最大可承受成本”其实很好理解，我们可以把人生的任何一次选择都看作是一次项目投资，投资就要付出成本，那么你能承担的最大成本是多少？经常有读者问我：“工作遇到瓶颈，我该转行吗？”“我该选择创业吗？”当然，这类问题也可以换做“我该继续读书还是选择就业？”“我该留在职场还是选择做全职妈妈？”……本质上，这些问题都是一样的，我的第一反应是：你的最大可承受成本是多少？每个人的风险偏好和风险承受能力不同，答案自然也就不同。【01】你的最大可承受成本是多少？2015年，33周岁的薛鸿涛已经是某公司的大区销售总监，工资加上奖金平均算下来月入6万。这个收入在北方某二线城市，绝对属于高收入。所以，当他决定加盟一家初创公司，每月拿着4800块生活补贴外加200块话费补贴时，家里人气得跳脚骂他“精神病”。那一年的6月，因为谈业务薛鸿涛认识了王承。此时的王承开了一家小广告公司，靠承接一些设计项目维持公司运转。公司虽然不大，但王承的梦想大，大得像一张悬在天上的馅饼儿。王承告诉薛鸿涛，他最崇拜的人是营销战略家杰克·特劳特。他将致力于在快消行业推行特劳特的定位理念，为客户提供品牌定位咨询服务。一开始，薛鸿涛觉得王承的牛皮吹得有点大。但当王承讲到自己公司在给快消品门店做设计时观察到客户的品牌定位需求，以及自己对文化溯源、品牌精髓、品牌理念的理解时，薛鸿涛才发现：王承是个人物！几次三番接触，薛鸿涛与王承惺惺相惜，创业的激情在薛鸿涛心中炸裂般翻腾。王承主动邀请薛鸿涛加入自己的团队，他说：“多少钱，你开价，不过我现在还给不起！我只能给得起每月四五千块的生活费，差额明年补齐，另外奉上一部分股权，算是咱兄弟俩合伙！”那时，我刚好在那个城市做一个项目，薛鸿涛跑来问我：“如果你是我，会怎么选？”我说：“我不是你，我的选择对于你而言没有任何参考价值。不过，你可以用最大可承受成本来评估一下。”我问薛鸿涛：“假设你和王承合伙创业失败，你还能找到和原来收入差不多的工作吗？”薛鸿涛说：“这并不难！”我问：“假设把这件事看作是一次项目投资，最大的风险是你损失掉的工资，一年半载三五十万成本，这个成本你能承受得了吗？”薛鸿涛肯定地说：“当然能！”我最后问：“那接下来你准备做出什么选择呢？”薛鸿涛一拍脑门，恍然大悟：“我知道了！”2019年，是薛鸿涛和王承合伙创业的第四年，公司业务发展迅猛。门店的设计业务为品牌咨询项目做了前期铺垫，品牌咨询项目又带来了新的设计订单，二者相互促进，形成了一个闭环。现在，不算股份分红，薛鸿涛的年收入也早已突破百万。后来，我把最大可承受成本的思路整理了一下，经常在生涯咨询时使用，启发来访者思考。那就是：当你要面对人生中一项重大抉择时，不妨大胆假设一下，如果这是一个投资项目，假设你彻底失败了，你能承受的最大成本是多少？这实际上是一个灵魂拷问，它能让你重新审视自己的风险承受能力。【02】为什么说学会拥抱风险是风险最小的事厌恶风险是人类的天性，但纵观人类发展史，没有任何一项进步，是不需要承担风险的。学会拥抱和驾驭风险，这才是成功的捷径，是对人类自我发展的最好保护。为什么这么说呢？1、世界是一个复杂系统，难以预测互联网的发展使人们的生活发生前所未有的变化，世界变成一个更加复杂的系统。以生涯咨询为例，以前我们称之为职业规划。但现在我觉得“规划”二字已经很难准确地表述它的定义。一是没有谁能替别人规划人生；二是互联网的发展到底能给人类的生活带来多大的变化，这是无法想象的。十年前，我们没有微信、没有短视频，而今，围绕着内容创业产生了多少新岗位？所以我更倾向于用“咨询”两个字，针对来访者的个人成长、职业困惑给出专业化的建议，而不是为你制定好五年、十年的规划。你可以按部就班走，但时代不会！那些影响人类发展的重大事件，从来都不是预测出来的。2、一件事情的影响力并非呈线性增长举个例子。你是抖音用户，你的粉丝不多，你每天都坚持发布某一领域的原创视频，一天稳定地涨几十、几百个粉丝，这就是线性增长。突然有一天，一个视频火了，一夜之间你涨了几万、十几万、甚至几百万粉丝，这就是非线性增长。这几年，非线性增长的个案越来越多。得益于互联网对个人影响力的放大，一款视频、一篇爆文、一次热门事件，你可能一下子就火了。我说这些，是想告诉你：过去，某些能从质的层面改变事物进程的事件发生的概率比较小，而今它的概率在不断增大。这一切，都指向四个字：不确定性。这就要求我们拥抱风险，敢于尝试。巴菲特说投资的秘诀就是：用四毛钱买一块钱的东西，那你肯定能获得高回报。但前提是，你愿意拿出那四毛钱，这其实就是一个撒网捞鱼的投资游戏。从商业回到个人，在这样的游戏中，一个重要的投资原则就是：你的最大可承受成本。你可以理解为：你用四毛钱买未来可能是一块钱的东西，这四毛钱可能会亏掉，但这个亏损是你能承受得起的。这样去探索，你才有成功的可能性。【03】怎样应对不可以预知的未来回到薛鸿涛的案例中来，你会发现，他之所以能收获那样一个完美的结局，是因为他有选择的权利。假设你是一个上班族，月入1万，有房贷有车贷，有父母要赡养，你敢冒这个风险吗？未必，因为薪水的大幅降低，意味着你可能连家都养不了。很多时候，不是我们不知道一件事情会给未来带来好的预期，而是我们真的没得选。那该怎么办呢？扩大选择权限！怎样才能扩大选择权限呢？分两步走。1、赛道优势什么是赛道优势？很简单，找到一个细分赛道去深耕，直到你拥有优势。判断优势有两个标准：一是相对优势，如果有100个人做这件事，你能打80-90分。你打到这个分数已经超越了业内的很多人，这就是相对优势。二是绝对优势，如果有100个人做这件事，你能打90分甚至以上，你打到这个分数，很有可能成为业内的头部，这就是绝对优势。你只有具备了赛道优势，才能进行接下来的一步：冠状跳跃。2、冠状跳跃什么是冠状跳跃呢？很简单，你在一个赛道内领先，跳到其他赛道仍然可能领先。举个例子。美洲雨林生活着一种猴子，叫卷尾猴。卷尾猴经常从一棵树冠跳到另一棵树冠，寻找食物或嬉耍，我把卷尾猴的跳跃动作叫做“冠状跳跃”。把这个词延伸到职场，意味着假设外界环境发生变化，需要你更换赛道，如果你原来就在赛道的头部，那么更换之后你仍然可能在头部。这实际上是一种优势的迁移。所以，很多时候为什么你没有选择权？因为你没有赛道优势，无论怎么更换赛道，都是在树底下盘桓，无法在树冠之间跳跃。机会偏爱有勇气的人，成功偏爱有底气的人。先打磨你的赛道优势，再观望冠状跳跃的时机！当面临选择时，如果最坏的结果你都能承受，那还犹豫什么？干，就完了！&lt;a 村长2019管理

2019-07-24 #刘润
禧人科技-任光辉 : 【村长2019管理日志】050期 5点起床，现在已经临近12点。今天村长想谈谈职业素养，换句话说，就是一个人在职场是否专业。 先来讲个故事。刚刚入职10天的销售总监小美，通过微信发来V2.0版本的宣传页文案。村长打开石墨文档链接，首先看到的是小美修改的地方，只有5处，并且是标题的优化。 细看文档，发现了几处低级错误。比如，1.0版本是写了7个一键功能。2.0版本把刚刚完成开发的销售PK加上，应当是8个一键功能。但是，文案上依旧是7个。 当下把小美喊到办公室进行沟通，指出问题，并明确表示，这样的低级错误应该消灭在我看到文稿之前。小美作为总监，应当发现文案编辑这个没有更正数量的错误。 今天下午，小美把做好的sop销售流程，又发到微信让我审阅。查看之后，依旧存在低级错误。比如，和最重要的决策者老板的话术，是复制的和人资洽谈内容。并没有把对话内容进行优化，HR二字竟然都没有修改。 连续两天出现同样的低级错误，村长实在无法忍受。晚上9点和股东进行通报，决定解除劳务合同，请小美另谋高就。发生在小美身上的故事，村长进行总结，发现这是个人职业化素养的问题，主要表现在三个方面。 1、算术错误和高数错误。1+1＝2是算术，如果做错就是低级的算术错误。高数是微积分，有一定的难度。比如因创新失败，而造成了损失，就是高数错误。管理者应当鼓励高数错误，杜绝算术错误。 小美连续二天在sop和宣传页文案上，都出现了低级的算术错误。作为总监级别，这是不可原谅的。 算术错误，连续出现，这是职业化素养不够。她缺少一个系统的工作方法，来避免自己犯下低级的算术错误的。 2、面谈＞电话＞微信＞QQ＞邮件。这是不同沟通方式的，沟通效果对比。面谈，是最高效的沟通方式。这也是，很多投资人在投过某个创业公司后，会要求创业公司搬到投资人所在地办公的原因。在一个城市，面谈的成本降低，沟通的效果却大幅度提升。 细心的读者应该观察到，小美两天发送的文件，都是通过微信进行传输，并在微信内进行文件内容的沟通。昨天，村长把她喊到办公室，并讲解了如何进行高效沟通。今天，依然进行微信沟通。当我回复四个字“来办公室”，才得以进行面谈。 同在一个300平方的办公室，为什么不来当面沟通？村长思考一下，感觉只有一个“懒”字可以说明问题。微信说几句，就解决问题了，这是“懒得动”。 放弃高效的面谈，选择容易产生歧义的微信，这是非职业化的体现。努力选择高效率的沟通方式，是一个职业化高手的行为习惯。 3、发现问题和解决问题。任何一个公司都会存在问题，不论他是多大规模，管理多么专业。任何一个人都会存在问题，不论他是多大年纪，知识多么渊博。企业没有问题，是在破产清算之后；人没有问题，是在驾鹤西去之后。 发现一个公司的问题并不难，难在是如何解决问题。村长公司的工作方法论是“事前打样，小干前通关，干中检查，事后复盘”。小美作为销售总监，是销售相关工作的第一责任人，打样建设sop是她的核心工作。 可是，小美却把优化sop的工作，推到了我这里。这是把问题这个猴子，扔到了我的背上。如果帮她进行优化，就是在替他解决问题。 可是，因为小美也刚刚入职10天，对公司的业务可能还不是很熟悉，怎么办？ 我解决的方法是，立即和一个销售模块的专家联系，确定了到杭州面谈的时间，希望通过专家的指导来帮助我们去解决问题。 发现问题不难，难得是解决问题，特别是能够善于借助外力来推动问题解决，这才是职业化的体现。 总结一下今天的内容，一个职业素养高的人，1、不犯低级错误，犯创新的高级错误；2、时刻保持高效率的沟通方式；3、不仅能够发现问题，并且能够解决问题，甚至是在自己能力不足时，借助外力解决问题。 当然，村长把职业化程度不高的小美招聘到公司，担任如此重要的岗位角色，我本人也有失察之责。分析犯这样错误的原因，是在面试时没有依据“技能宝典”，设计好面试问题，从而把小美放到了一个并不适合她的位置。 最后，村长给小美分析了她的优势，提出了走“商务洽谈”这个职业发展路线。算是对自己犯下的错误，进行的亡羊补牢吧。2019-07-24点赞数:16评论区：禧人科技-任光辉 : 有没有一种可能，是我错了。是我的管理水平太差，职业素养不足呢？ 123 : 最后一段话，真没有几个管理者能做到。[强] 琳婧 : 我不认同你的关点。选人、育人、用人上面选人在最前面并且很重要。 蒋杰 : 跟管理风格有关，揉不揉的进沙子，有没有同理心，好多话boss也得跟下属多聊心的，别端着，你不说他们怎么知道呢，毕竟老板主要是给员工当服务员，帮助他们成功的 周锋 : 为了凸现管理而管理。 王天海 : 把人放在合适的位置上，这可能是关键，说不定她是一个厉害的销售，但不是一个厉害的销售总监，其实也看怎么来用，公司对销售总监工作的定位
